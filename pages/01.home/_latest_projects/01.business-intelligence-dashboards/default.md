#### Problems
* Lack of graphs native to the Netsuite platform.
* Desire to display department specific graphs on video display boards on department walls.

#### Solution
* Consulted with company stakeholders and built out department business intelligence dashboards.
* Built API integrations to gather CRM, accounting & project data on a periodic basis and store in database tables.
* Built out MySQL table views to correlate or summarize data between multiple record types.
* Built out cron jobs to turn MySQL data into JSON objects for retrieval from dashboards.
* Built out graphic dashboards using pie charts, bar charts, gauge charts, tables, heat maps, and other charts to display relevant BI to company stakeholders with regular ajax regreshing of data.

#### Skills Used
Web Development, SOAP, API Integration, API, Bootstrap, Google Charts, MySQL, Netsuite Development, Highcharts, Chart.js
