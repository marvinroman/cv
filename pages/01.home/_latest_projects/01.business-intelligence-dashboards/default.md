#### Workflow
* Consulted with company stakeholders and built out department business intelligence dashboards.
* Built API integrations to company ERP system to pull accounting, CRM & project information to gather new data regularly.
* Built out MySQL table views to correlate data between multiple record types.
* Built out cron jobs to turn MySQL data into JSON objects for retrieval from dashboards.
* Built out graphic dashboards using pie charts, bar charts, gauge charts, tables, heat maps, and other charts to display relevant BI to company stakeholders.

#### Skills Used
Web Development, SOAP, API Integration, API, Bootstrap, Google Charts, MySQL, Netsuite Development, Highcharts, Chart.js
