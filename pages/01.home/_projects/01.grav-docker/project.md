A Docker built specifically for use with [Grav CMS](https://getgrav.org/). Forked from [ric_harvey/nginx-php-fpm](https://gitlab.com/ric_harvey/nginx-php-fpm). The docker has over 30 configuration variables that customize & automate the setup of the docker container.
